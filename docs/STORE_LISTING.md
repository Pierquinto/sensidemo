# Store Listing Draft

## Name

SensiDemo

## Short description

Blur sensitive images and text per domain while recording product demos.

## Detailed description

SensiDemo helps product teams, founders, sales teams, support teams, and creators record demos without accidentally exposing private user data.

Use it to blur profile photos, names, roles, account information, customer lists, CRM rows, contact cards, and other sensitive page content.

Features:

- Manual ON/OFF toggle per tab.
- Per-domain rules.
- Click-to-blur selection.
- Recursive rules for lists, grids, cards, and rows.
- Per-rule blur intensity from 1 to 10.
- Local-only storage.
- No backend, analytics, or telemetry.

SensiDemo is a visual aid for screen recordings. It does not remove data from the page DOM or developer tools.

## Privacy practices summary

SensiDemo stores configuration locally in the browser. It does not collect, transmit, sell, or share personal data.
