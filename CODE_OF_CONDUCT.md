# Code of Conduct

SensiDemo follows the Contributor Covenant Code of Conduct.

## Our pledge

We are committed to making participation in this project a respectful, harassment-free experience for everyone.

## Expected behavior

- Be kind and constructive.
- Assume good intent.
- Focus feedback on the work, not the person.
- Respect privacy and security concerns.

## Unacceptable behavior

- Harassment, threats, or discriminatory language.
- Sharing private information without permission.
- Publishing security issues before maintainers have had a reasonable chance to respond.

## Enforcement

Project maintainers may remove comments, close issues, reject pull requests, or block participants who violate this code of conduct.
