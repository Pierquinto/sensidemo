# Changelog

## 0.1.0

- Initial source-available noncommercial release.
- Manual per-tab blur toggle.
- Per-domain rule storage.
- Click-to-blur selection.
- Single-element and recursive list/grid rules.
- Per-rule blur intensity from 1 to 10.
- Flicker-resistant re-blurring for dynamic DOM updates.
- Demo page for local testing.
